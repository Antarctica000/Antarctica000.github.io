2.1.1 我方坦克发射子弹

  1. myTank有子弹的对象，按下J时就启动一个发射行为（线程）
  2. MyPanel需不停重绘才能让子弹不停移动，形成一个设计效果
  3. 子弹移动到面板边界时（或打到敌人）就应该销毁（销毁线程）
  4. 问题二：怎么实现连发子弹？怎么实现坦克一边走一边发射子弹？

2.1.2 敌方坦克发射子弹

  1. 敌人坦克类使用Vector保存多个Shot
  2. 每创建一个敌坦对象就给该对象初始化一个Shot对象，同时启动Shot
  3. 绘制敌人坦克时，遍历Vector并绘制所有子弹，当子弹isLive为false时从Vector移除
  4. 问题三：为什么敌人坦克发射子弹速度那么快？

问题三解决：：问题三原因是地方子弹启动线程的位置不对，应该是创造之后就立马启动。 之前的线程启动写在画出坦克方法里面了,与paint产生了一些奇怪的反应 导致线程越开越多 速度疯狂增加

2.2 坦克被击中

2.2.1 敌方坦克消失

  1. 判断子弹是否击中敌人坦克方法
  2. 重绘时进行判断
  3. 我方坦克子弹未打出时注意空指针异常
  4. 问题四：为什么我方坦克发射子弹消灭敌人之后 在原来敌人的位置子弹无法穿透？
  5. 问题五：判断能否放在别的敌方？比如绘制坦克前？

问题四解决：其实很简单。敌方坦克“尸位素餐”。虽然敌方坦克没有重绘出来，但是”尸体“并未消失。所以只需remove被击中的敌方坦克即可。
